I coded a very crude 2D turbulent fluid simulation based on the <a href="http://en.wikipedia.org/wiki/Two-dimensional_point_vortex_gas" target="_blank">point vortex</a> method. I was inspired by <a href="http://madebyevan.com/webgl-water/" target="_blank">this</a> amazing real-time water simulation, which is why I did it in Javascript and HTML5 canvas. I'm going to eventually make it a smoke simulation or do something like <a href="http://www.math.mcgill.ca/~applsem/no-beta.gif" target="_blank">this</a>. Anyways, <a href="http://www.mshang.ca/vortex.html" target="_blank">here</a> it is. All the source code is included in the .html file (use your browser's "View Source").
