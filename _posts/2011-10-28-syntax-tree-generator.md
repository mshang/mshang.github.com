<a href="http://mshang.github.com/syntree" target="_blank">Here</a>'s a small Javascript app for generating syntax trees. You will need an up to date browser to use it. In fact, I've only tried it in Google Chrome. Why make another tree app when there are tons already out there? This one is lightweight and portable. It works entirely within the browser (i.e. no server-side). This means that you can use it without an internet connection and without installing anything. One major goal is to add support for movement arrows.
